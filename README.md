# 시니어 추억 기록 플랫폼

시니어를 위한 따뜻한 추억 기록 웹서비스입니다.

## 주요 기능

- 📝 **추억 등록**: 텍스트, 사진, 음성, 영상으로 추억을 기록할 수 있습니다
- 📅 **타임라인 뷰**: 시간순으로 추억을 감상할 수 있습니다
- 📸 **앨범**: 사진과 영상만 모아서 볼 수 있습니다
- 👨‍👩‍👧‍👦 **가족 구성원 관리**: 가족 구성원을 추가하고 관리할 수 있습니다
- ❤️ **좋아요 & 댓글**: 가족 구성원들과 소통할 수 있습니다
- 🏷️ **태그 필터**: 태그로 추억을 필터링할 수 있습니다

## 기술 스택

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Shadcn UI**
- **로컬 스토리지** (데이터 저장)

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 3. 빌드

```bash
npm run build
npm start
```

## 프로젝트 구조

```
/app
  /timeline          # 타임라인 페이지
  /memory
    /new             # 추억 등록 페이지
    /[id]            # 추억 상세 페이지
  /album             # 앨범 페이지
  /family            # 가족 구성원 관리 페이지
/components
  /ui                # Shadcn UI 컴포넌트
  BottomNav.tsx      # 하단 네비게이션
  MemoryCard.tsx    # 추억 카드 컴포넌트
/lib
  db.ts              # 데이터베이스 유틸리티
  utils.ts           # 유틸리티 함수
```

## 주요 특징

### 시니어 친화적 UI
- 큰 글씨 (기본 18px)
- 큰 버튼 (최소 높이 64px)
- 넓은 여백
- 직관적인 아이콘

### 따뜻한 디자인
- 오렌지·핑크 톤 컬러 팔레트
- 부드러운 그라데이션 배경
- 둥근 모서리와 그림자 효과

### 반응형 디자인
- 모바일 중심 레이아웃
- 하단 탭 네비게이션

## 데이터 저장

현재는 브라우저의 로컬 스토리지를 사용합니다. 프로덕션 환경에서는 Supabase나 다른 데이터베이스로 교체할 수 있습니다.
